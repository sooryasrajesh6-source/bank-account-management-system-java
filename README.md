# Bank Account Management System (Java)

A simple console-based Bank Account Management System developed using Java and Object-Oriented Programming (OOP) concepts.

## Features

- Create a bank account
- Deposit money
- Withdraw money
- Check account balance
- View account details
- Menu-driven console application

## Technologies Used

- Java
- OOP Concepts
- Scanner Class
- Command Line Interface (CLI)
